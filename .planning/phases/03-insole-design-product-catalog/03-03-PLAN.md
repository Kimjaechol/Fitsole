Create `services/measurement/app/insole/optimizer.py` implementing per D-01, D-02, D-04:

1. `calculate_optimal_arch_height(navicular_height, midfoot_pressure_ratio, body_weight, foot_length) -> float`:
   - Base: 35mm (flat, navicular<15), 42mm (normal), 50mm (high, navicular>25) per D-01
   - Pressure adjustment: (0.5 - midfoot_pressure_ratio) * 10.0
   - Weight adjustment: (body_weight - 70) * 0.05
   - Length scaling: foot_length / 260.0
   - Clamp to [25.0, 60.0], round to 1 decimal

2. `calculate_optimal_heel_cup_depth(heel_peak_pressure, pronation_degree, age, activity_type) -> float`:
   - Base: 20mm per D-02
   - Pressure adj: +5 if >300kPa, +2.5 if >200kPa
   - Pronation adj: abs(pronation_degree) * 0.5
   - Age adj: max(0, (age-40)*0.1)
   - Activity adj: running=+3, standing=+2, daily=0
   - Clamp to [15.0, 35.0], round to 1 decimal

3. `generate_insole_design(input: InsoleDesignInput, measurements: dict, pressure_data: dict, gait_data: dict) -> InsoleDesignResult`:
   - Extract navicular_height from measurements (use archHeight as proxy)
   - Extract midfoot_pressure_ratio from pressure_data heatmap (ratio of midfoot zone intensity to total)
   - Extract heel_peak_pressure from pressure_data high_pressure_zones
   - Extract pronation_degree from gait_data
   - Call arch height and heel cup functions
   - Calculate remaining DesignParams: eva_cushion=10mm, forefoot_flex from gait data, medial/lateral post from pronation
   - Apply hardness map
   - Return InsoleDesignResult

4. `recommend_shoe_size(foot_length_mm, foot_width_mm, product_sizes) -> dict`:
   - Standard Korean sizing: foot_length (mm) -> shoe size (e.g., 260mm -> 260)
   - Find closest match in product_sizes by lastLength/lastWidth
   - Return recommended size and fit notes (narrow/wide)

Create `services/measurement/app/insole/hardness_mapper.py` per D-03:
