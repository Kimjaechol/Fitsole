1. Create `src/app/api/checkout/confirm/route.ts` — POST endpoint per D-10, D-11:
   - Accepts: `{ paymentKey: string, orderId: string, amount: number }` from Toss redirect
   - Verify payment server-side by calling Toss Payments confirm API:
     ```typescript
     const response = await fetch('https://api.tosspayments.com/v1/payments/confirm', {
       method: 'POST',
       headers: {
         'Authorization': `Basic ${Buffer.from(process.env.TOSS_SECRET_KEY + ':').toString('base64')}`,
         'Content-Type': 'application/json',
       },
       body: JSON.stringify({ paymentKey, orderId, amount }),
     });
     ```
   - If Toss returns success: update orders table — set status="paid", paymentKey, paymentMethod (from response.method), paidAt=now()
   - If Toss returns error: update orders table — set status="cancelled", return error
   - After successful confirmation: call sendOrderConfirmationEmail() (from Task 2)
   - Return `{ success: true, orderNumber, orderId }` or `{ success: false, error }`
   - Note: orderId here is the orderNumber we sent to Toss (FS-YYYYMMDD-XXXX), need to look up by orderNumber
