**Extend `src/lib/db/schema.ts`** -- add 4 new tables per RESEARCH.md code example:

1. `scanStatusEnum` pgEnum: 'uploading', 'processing', 'completed', 'failed'
2. `footSideEnum` pgEnum: 'left', 'right'

3. `footScans` table: id (uuid PK), userId (uuid FK to users.id cascade), footSide (footSideEnum), status (scanStatusEnum default 'uploading'), videoUrl (text nullable), modelUrl (text nullable), qualityScore (real nullable), qualityLabel (text nullable), processingStage (text nullable), errorMessage (text nullable), createdAt (timestamp defaultNow), completedAt (timestamp nullable)

4. `footMeasurements` table: id (uuid PK), scanId (uuid FK to footScans.id cascade), footLength (real), ballWidth (real), instepHeight (real), archHeight (real), heelWidth (real), toeLength (real), createdAt (timestamp defaultNow)

5. `gaitAnalysis` table: id (uuid PK), scanId (uuid FK to footScans.id cascade), gaitPattern (text), ankleAlignment (text), archFlexibilityIndex (real), walkingVideoUrl (text nullable), landmarksData (jsonb nullable), createdAt (timestamp defaultNow)

6. `pressureDistribution` table: id (uuid PK), scanId (uuid FK to footScans.id cascade), heatmapData (jsonb), highPressureZones (jsonb nullable), inputWeight (real nullable), inputGender (text nullable), inputAge (real nullable), createdAt (timestamp defaultNow)

Import `real`, `jsonb`, `pgEnum` from 'drizzle-orm/pg-core' in addition to existing imports.

**Create `src/app/api/scan/create/route.ts`** -- POST endpoint:
- Check auth session via auth() from '@/lib/auth'
- Accept body: { footSide: 'left' | 'right' }
- Validate with Zod: z.object({ footSide: z.enum(['left', 'right']) })
- Insert into footScans with userId from session, status 'uploading'
- Return 201 with { id, footSide, status }
- Return 401 if not authenticated

**Create `src/app/api/scan/status/[id]/route.ts`** -- GET endpoint:
- Check auth session
- Query footScans where id = params.id AND userId = session.user.id (IDOR prevention per security domain)
- Return { id, status, processingStage, qualityScore, qualityLabel, errorMessage }
- Return 404 if not found or not owned by user

**Create `src/app/api/scan/results/[id]/route.ts`** -- GET endpoint:
- Check auth session
- Query footScans JOIN footMeasurements JOIN gaitAnalysis JOIN pressureDistribution where footScans.id = params.id AND userId = session.user.id
- Return full ScanResult object matching the TypeScript type from types.ts
- Return 404 if not found, 401 if not authenticated
