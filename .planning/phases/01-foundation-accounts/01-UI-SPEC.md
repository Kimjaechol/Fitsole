## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, input internal padding |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, form group gaps |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Landing page section breaks |
| 3xl | 64px | Page-level vertical rhythm (hero top/bottom padding) |

Exceptions: Bottom tab bar height is 56px (not a multiple of 8 but standard mobile nav height for 44px touch targets + 12px safe area).

| Label | 14px | 600 (SemiBold) | 1.4 |
| Heading | 24px | 700 (Bold) | 1.2 |

Note: Pretendard loaded via CDN (`https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable.min.css`) or self-hosted from `static/fonts/`. Variable font preferred for weight flexibility within the declared 400/600/700 range.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#FFFFFF` (white) | Page background, content surfaces |
| Secondary (30%) | `#F8FAFC` (slate-50) | Card backgrounds, profile section bg, nav bg, input bg |
| Accent (10%) | `#2563EB` (blue-600) | Primary CTA buttons, active tab indicators, links, focus rings |
| Accent Secondary | `#059669` (emerald-600) | Success states, health/science trust indicators, measurement process icons |
| Destructive | `#DC2626` (red-600) | Account deletion confirmation, error alerts |
| Border | `#E2E8F0` (slate-200) | Card borders, input borders, dividers |

Accent reserved for: Primary CTA buttons ("내 발에 맞는 신발 찾기", "시작하기"), active bottom tab icon + label, text links, form input focus rings, login/signup submit button.

Accent secondary reserved for: Landing page measurement process step icons, success toast/banner after signup, health-related badge/pill elements.

## Interaction Patterns

### Bottom Tab Bar
- Active tab: accent color icon + label, inactive tabs: muted text color
- Scan tab visually differentiated (larger icon or accent background circle) to draw attention
- Tap feedback: scale(0.95) on press, 100ms duration

### Auth Forms
- Input validation: inline error messages below each field in destructive color
- Password: minimum 8 characters, show/hide toggle (Eye/EyeOff Lucide icons)
- Submit button: full width, 48px height, disabled state at 50% opacity while loading
- Loading state: spinner icon replaces button text during API call

### Empty States
- Centered layout: illustration placeholder (64px icon from Lucide) + heading + body + CTA button
- CTA button uses primary accent style

### Landing Page Sections
- Hero: full viewport height on mobile, vertically centered content
- Value columns: 3-column grid on desktop, single column stack on mobile
- Process preview: horizontal 3-step flow with connecting lines on desktop, vertical on mobile
- Bottom CTA: accent background section, white text, centered CTA button

### Page Transitions
- Use Framer Motion `fadeIn` for page transitions: opacity 0 to 1, 200ms ease-out
- Tab switching: no animation, instant swap
