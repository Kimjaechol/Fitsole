# 신발 내부 스캔 + 병합 SOP

> **최종 개정일:** 2026-04-11
> **대상 역할:** 측정팀 (내부 운영자)
> **목적 문서:** 이 SOP는 작업대에서 단독으로 사용할 수 있도록 작성되었습니다. 외부 문서를 열지 않고도 한 켤레의 신발에 대한 Revopoint + 알지네이트 하이브리드 캡처와 관리자 대시보드 병합까지 전 과정을 수행할 수 있어야 합니다.
> **참고값(*)** 표기는 초기 권장값이며, 실측 데이터가 누적되면 조정할 수 있습니다.

---

## 목적

FitSole는 고객의 발 측정 데이터와 매칭되는 "맞춤 인솔 + 신발 세트"를 판매한다. 이때 신발 모델마다 내부 치수(내부 길이, 너비, 힐컵 깊이, 아치 지지점, 토박스 부피, 발등 여유)가 다르며, 동일 사이즈라도 브랜드/모델 간 최대 11mm 수준의 편차가 존재한다. 본 SOP는 **Revopoint MINI 2 광학 스캐너**로 신발 내부를 직접 촬영한 부분 메시와 **알지네이트 캐스트**로 본뜬 외부 메시를 한 쌍으로 획득하고, 이를 관리자 대시보드(`/admin/dashboard/shoe-merge`)에서 병합하여 인솔 설계의 입력이 되는 통합 치수(resolved_dimensions)를 산출하는 절차를 정의한다.

## 사전 체크리스트

- [ ] 신발 내부 이물질을 부드러운 브러시로 제거, 내부가 젖어있지 않은지 확인
- [ ] 작업대 조명이 ≥ 500 lux 이며 간접광인지 확인(직사광 금지)
- [ ] Revopoint 소프트웨어 최신 버전, 월 1회 캘리브레이션 이력 확인
- [ ] 알지네이트 분말과 물 온도가 실온(20–25°C)에 맞추어져 있는지 확인 (차가운 물일수록 경화 지연)
- [ ] 기준 마커/패턴 타겟이 신발 입구 주변의 비손상 위치에 부착되어 있는지 확인
- [ ] 샘플 신발에 고유 `scanId` 발급(UUID) — 이후 모든 산출물 파일명의 접두사로 사용
- [ ] 일회용 장갑, KF94 마스크, 비닐 라이너 재고 확인
- [ ] 작업대 턴테이블이 수평인지, 무게 중심이 회전 축과 일치하는지 확인

## Phase 1 — Revopoint 부분 내부 스캔 (3~5분)

1. 신발 입구가 위로 향하도록 **턴테이블 중앙**에 고정한다. 필요 시 논슬립 패드 사용.
2. **LED 링 조명 ON**. 신발 내부 입구에서 반사광이 직접 카메라로 돌아오지 않도록 조명 각도를 조정한다.
3. Revopoint 소프트웨어에서 **"Body" 모드**, resolution **"High"** 를 선택한다. *(참고값)*
4. 입구 기준으로 **±30° 궤적** 내에서 천천히 회전시키며 내부 상부와 측벽을 커버한다.
5. 토박스 안쪽 끝과 힐컵 뒷벽 등 **사각지대**는 무리하게 찍지 않는다. 해당 영역은 Phase 2 캐스트가 담당한다.
6. 스캔을 `.stl` 형식으로 저장한다. 파일명은 `{scanId}-revopoint.stl` 로 통일한다. 허용 포맷은 `.stl / .obj / .ply / .gltf / .glb` 이다.

**Coverage 권고:** 입구에서 들어가는 영역의 표면 커버리지 **≥ 60%** *(참고값)*. 이보다 낮으면 Phase 1을 재촬영한다.

**자주 발생하는 실수:**

- 입구에 손가락이나 케이블이 비쳐 메시에 노이즈로 포함됨 → 삼각대 + 원격 셔터 사용
- 실내 형광등의 60Hz 깜빡임 → LED 링 조명만 사용, 형광등 소등
- 신발이 턴테이블에서 미끄러짐 → 무반사 논슬립 패드 추가

---

## Phase 2 — 알지네이트 캐스트 생성 (3~5분 경화)

1. 신발 내부에 **비닐 라이너**를 밀착 삽입한다. 라이너는 이형제 역할을 하며, 라이너가 없으면 알지네이트가 신발 내부에 달라붙어 파손될 수 있다.
2. 알지네이트 분말과 물을 **제품 권장 비율**로 계량한다. 일반 치과용은 1 : 0.9 (w/w) 수준 *(참고값, 제품 라벨이 우선)*.
3. **60초 이내**에 크림 질감이 될 때까지 스파츌라로 균질 혼합한다. 혼합이 끝난 시점에 스톱워치를 시작한다.
4. 주사기에 혼합물을 채워 **뒤꿈치 → 아치 → 토박스** 순으로 주입한다. 이 순서를 지키면 공기 갇힘을 최소화할 수 있다.
5. 제품 권장 경화 시간만큼 대기한다(일반적으로 3–5분). 대기 중 신발은 움직이지 않는다.
6. 경화 확인 후 **비닐 라이너와 함께** 캐스트를 천천히 추출한다. 신발 내부에 잔여물이 없는지 브러시로 확인한다.

**안전 주의:**

- 알지네이트 분말 혼합 단계에서 반드시 KF94 마스크 착용 (분진 흡입 방지)
- 라이너 없이 직접 주입 금지 — 신발 소재(특히 스웨이드, 메쉬)가 손상될 수 있다
- 라텍스 알레르기 작업자는 니트릴 장갑 + 분말 없는 알지네이트 사용

---

## Phase 3 — 캐스트 외부 스캔 (2~3분)

1. 추출한 캐스트를 턴테이블 위에 세팅한다. 뒤꿈치 쪽을 아래로 두고 수직으로 안정되게 위치시킨다.
2. Revopoint를 **360° 회전 스캔** 모드로 전환한다. 캐스트 표면이 반사율이 낮으므로 마커 스프레이는 불필요하다.
3. 캐스트의 **상단 → 중단 → 하단** 세 레이어를 모두 커버하도록 수직 각도를 바꿔 가며 2–3 패스 촬영한다.
4. `.stl` 형식으로 저장한다. 파일명은 `{scanId}-cast.stl` 로 통일한다.
5. 스캔 직후 캐스트는 별도 트레이에 라벨링(`scanId`, 촬영일시)하여 **24시간 보관** 후 폐기한다. 재병합 요청이 있을 수 있다.

---

## Phase 4 — 관리자 대시보드 병합

1. 관리자 권한 브라우저에서 `/admin/dashboard/shoe-merge` 로 이동한다.
2. 업로드 양식에서 **Revopoint 파일** 선택 → **알지네이트 캐스트 파일** 선택 → **scanId** 입력(Phase 0에서 발급한 UUID를 그대로 사용).
3. **"병합 실행"** 버튼을 눌러 결과를 기다린다. 파일당 최대 100MB 까지 업로드할 수 있다.
4. 결과 화면에서 다음 지표를 확인한다:
   - **alignment_rmse** (mm): 녹색 < 1.0, 노랑 1.0–2.0, 빨강 ≥ 2.0
   - **overlap_percentage** (%): 녹색 > 70, 노랑 50–70, 빨강 ≤ 50
   - **discrepancy_count**: 녹색 0, 노랑 1–2, 빨강 ≥ 3
   - **resolved_dimensions** 값 spot-check: 내부 길이가 물리적으로 그럴듯한 범위(예: 230 mm 사이즈라면 240–260 mm)인지 육안 확인
   - **resolution_report**: 각 치수별 해결 방법(method)과 신뢰도(confidence)
   - **경고(warnings) 목록**: 비어 있으면 이상 없음. 항목이 있으면 다음 장의 트러블슈팅 참조.
5. 필요 시 **"사이즈 그레이딩 실행"** 버튼을 눌러 앵커 사이즈(기본 270mm)에서 한국 사이즈 **230–290mm** 예측값을 확보한다. validation_warnings 가 있으면 파트너 승인 전에 확인한다.
6. 모든 값이 합격이면 결과 JSON을 `/var/fitsole/scans/{scanId}/merge-result.json` 경로에 백업한다.

## 품질 기준 (Quality Gates)

| 지표                 | 합격          | 주의             | 재스캔 필요    |
| -------------------- | ------------- | ---------------- | -------------- |
| alignment_rmse       | < 1.0 mm      | 1.0 – 2.0 mm     | ≥ 2.0 mm       |
| overlap_percentage   | > 70 %        | 50 – 70 %        | ≤ 50 %         |
| discrepancy_count    | 0             | 1 – 2            | ≥ 3            |

> 위 값은 현재 `ShoeMergeClient` UI와 동일한 기본 임계값이다. 향후 실측 데이터가 누적되어 임계값을 변경해야 할 경우, **UI 상수(`RMSE_THRESHOLDS`, `OVERLAP_THRESHOLDS`, `DISCREPANCY_THRESHOLDS`)와 본 SOP의 표를 반드시 동시에** 수정한다.

## 재스캔 기준

다음 경우 중 하나라도 해당되면 재스캔을 수행한다.

- `alignment_rmse` ≥ 2.0 mm → Phase 1과 Phase 3을 모두 재스캔
- `alignment_rmse` 1.0–2.0 mm + `discrepancy_count` ≥ 3 → Phase 3(캐스트 스캔) 우선 재스캔
- `overlap_percentage` ≤ 50 % → Phase 1 커버리지 부족 가능성 → Phase 1 재스캔
- resolved_dimensions 중 **null 또는 물리적으로 불가능한 값**이 2개 이상 → Phase 1 + Phase 2 모두 재수행
- resolution_report 에서 같은 치수 항목의 confidence < 0.50 이 2개 이상 → Phase 1 재스캔
- Phase 2에서 캐스트 추출 시 **찢어짐/결손**이 있었던 경우 → Phase 2 + Phase 3 재수행
- Phase 2 캐스트 안에 **에어포켓** 육안 확인 → Phase 2 + Phase 3 재수행
- 병합 warning 에 "heel_cup_missing" 또는 "toe_box_missing" 이 포함된 경우 → Phase 1 + Phase 2 재수행

## 안전 및 위생

- **분진 노출 관리:** 알지네이트 분말 혼합 시 KF94 이상 마스크 필수. 국소 배기 또는 환기가 가능한 공간에서 수행한다.
- **라텍스 알레르기:** 라텍스 장갑 대신 니트릴 장갑 사용. 라텍스 프리 알지네이트 재고가 있을 경우 우선 사용.
- **장비 청소:** Phase 2 종료 후 보울과 스파츌라는 **3분 이내**에 물로 1차 세척, 경화 후에는 스크레이퍼로 제거. 알지네이트는 배수구에 버리지 않는다(배관 막힘 위험).
- **신발 오염 관리:** 캐스트 추출 후 라이너에 잔여물이 묻어 있을 수 있으므로, 신발 내부를 부드러운 브러시로 점검하고 24시간 자연 건조 후 고객 반송.
- **개인 위생:** 혼합 전후 손 씻기, 장갑은 Phase 단위로 교체.
- **폐기물 처리:** 사용한 캐스트, 라이너, 장갑은 일반 폐기물로 분류(알지네이트는 생분해성). 상업 폐기 지침에 따름.
